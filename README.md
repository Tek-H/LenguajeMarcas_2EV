## 📖 DESCRIPCIÓN:
Este proyecto de GitHub consiste en una reestructuazión y mejora de la página web del Arsenal FC. El objetivo principal es modernizar el diseño y mejorar la organización del contenido
## 📂 ESTRUCTURA:
El proyecto se ha organizado en carpetas. Una carpeta para las imágenes donde se organizan por páginas y por temas. Por otro lado, las páginas se guardan en carpetas con su mismo nombre o en aquellas que se relacionan directamente con ellas.
## COMO ABRIR:
Para poder visualizar la web es necesario hacer `git clone (url del repositorio)` y posteriormente abrir cualquiera de los archivos HTML.
